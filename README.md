# Colon Cancer Detection using Machine Learning

## 📌 Description
This project focuses on detecting colon cancer using high-dimensional gene expression data. The model is designed to handle challenges such as class imbalance and high feature dimensionality to improve prediction accuracy.

## 🚀 Key Features
- Ensemble learning model combining SVM, KNN, and Random Forest
- Handling imbalanced dataset using SMOTE
- Feature selection techniques to reduce dimensionality
- Optimization using Genetic Algorithm
- High accuracy prediction performance

## 🛠️ Techniques Used
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Random Forest
- SMOTE (Synthetic Minority Over-sampling Technique)
- Feature Selection (Chi-Square, Symmetrical Uncertainty, CFS)
- Genetic Algorithm (for weight optimization)

## 📊 Results
- Achieved up to **100% accuracy** on balanced dataset
- Achieved approximately **94%+ accuracy** across different models
- Significant improvement over baseline models
